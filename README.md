# 🌿 EcoPlant - Інтернет-магазин кімнатних рослин

Навчальний проєкт, розроблений в рамках лабораторної роботи №1 з дисципліни «WEB-орієнтовані технології. Backend розробки». 

Проєкт представляє собою клієнтську частину (фронтенд) адаптивного веб-застосунку для продажу кімнатних рослин із закладеною базою для майбутньої інтеграції серверної частини (бекенду).

## 🚀 Функціонал
* **Семантична верстка:** Використання сучасних тегів HTML5 (`<header>`, `<main>`, `<footer>`, `<article>`).
* **Адаптивний дизайн:** Коректне відображення на десктопних, планшетних та мобільних пристроях завдяки CSS Flexbox, Grid та медіа-запитам.
* **Інтерактивність:** Робоче модальне вікно кошика та адаптивне "бургер-меню" (реалізовано за допомогою ванільного JavaScript).
* **Багатосторінковість:** Проєкт містить три пов'язані сторінки (Головна, Каталог, Догляд).

## 🛠 Інструкція щодо запуску
Проєкт не потребує встановлення додаткових залежностей або локального сервера для базового перегляду фронтенд-частини.

1. Зклонуйте цей репозиторій на свій комп'ютер за допомогою команди:
   `git clone [посилання_на_ваш_репозиторій]`
2. Перейдіть у папку проєкту.
3. Відкрийте файл `index.html` у будь-якому сучасному веб-браузері (Chrome, Firefox, Safari тощо).
*Примітка: Для зручнішої розробки рекомендується використовувати розширення Live Server для VS Code.*

## 📁 Структура проєкту
* `/assets/` - графічні ресурси (зображення рослин, фони).
* `/css/` - каскадні таблиці стилів (`style.css`).
* `/js/` - скрипти для забезпечення інтерактивності (`app.js`).
* `*.html` - сторінки веб-застосунку.
